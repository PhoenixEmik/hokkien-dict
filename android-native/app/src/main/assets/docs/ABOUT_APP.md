# 台語辭典 / Taigi Dict

台語辭典是以離線使用為核心的台語與華語辭典 App。

Taigi Dict is an offline-first Taiwanese Hokkien and Mandarin dictionary app.

## Product Identity

- App name: 台語辭典
- Android application ID: org.taigidict.app
- Official domain: https://taigidict.org
- Production assets: https://app.taigidict.org/assets/

## Core Features

- Search Taiwanese headwords, Tailo romanization, and Mandarin definitions
- Save and reopen bookmarks
- Download and play offline word/example audio archives
- Support Traditional Chinese, Simplified Chinese, and English UI
- Adjust theme and reading text size
